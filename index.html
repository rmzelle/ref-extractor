Reference Extractor
Extract Zotero and Mendeley references from Microsoft Word files

Reference Extractor is a tool to extract Zotero and Mendeley references from Microsoft Word .docx documents, so that they can be imported into reference management software. For documents created with Zotero, the references must be stored as "Fields", not "Bookmarks" (see the Zotero document preferences). For references inserted with Zotero 3.x or 4.x, the "Store references in document" option must also have been checked.

Extracted references can be saved in the CSL JSON and BibTeX formats. The CSL JSON format is lossless, and can be imported into Zotero by clicking the "Download" button and importing the downloaded JSON file via the "Import..." option in Zotero's File menu, or by clicking the "Copy to Clipboard" button and using Zotero's "Import from Clipboard" option.

I care about your privacy. With this tool, your document never leaves your computer (your internet browser performs the extraction).
